# WandForm Execution Plan (Post-Research)

## Product North Star

Ship the fastest path to:

- realtime collaborative form creation,
- fair and transparent pricing,
- multi-tenant custom-domain SaaS on Cloudflare.

## Build Tracks

## Track A — Core Product (Builder + Runtime)

- Improve form schema editor UX
- Add collaboration comments + change timeline
- Add anti-spam and abuse controls

## Track B — Data + Analytics

- Capture events: view, start, complete, submit errors
- Add drop-off summary by field
- Add per-form conversion dashboard endpoint

## Track C — SaaS Platform

- Tenant API
- Domain onboarding API (custom hostnames metadata flow)
- RBAC basics for teams/workspaces

## Track D — GTM + Validation

- Public comparison pages
- Waitlist + migration-interest capture
- 10 design-partner onboarding process

## 2-Week Delivery Slice (immediate)

1. Turnstile/honeypot/rate limits
2. Analytics event table + API
3. Comments on builder realtime channel
4. Typeform JSON importer (first pass)
5. Landing page copy focused on pricing fairness + collaboration

## Success Metrics

- Time-to-first-form: < 3 minutes
- Free-to-Pro conversion (pilot): > 5%
- Weekly active builder collaborators: increasing week-over-week
- Submission success rate: > 99%
- P95 form load latency: < 300ms (edge regions)
